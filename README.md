# Guarded Mind Wellness Center

A professional website for Guarded Mind Wellness Center, a psychiatric private practice serving Sykesville and Westminster, Maryland.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## Build for Production

```bash
npm run build
npm run preview
```

## Customization

- **Clinician name**: Update `[Your Name]` in `src/components/Bio.jsx`
- **Psychology Today link**: Replace the placeholder URL in `src/components/Contact.jsx`
- **Profile photo**: Replace the avatar placeholder in `src/components/Bio.jsx` with an `<img>` tag

## Tech Stack

- React 18
- Vite 5
